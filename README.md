# an6474

Hugo's theme

## install

```
$ git clone https://github.com/va2577/an6474.git theme/an6474
```

## stylus

```
$ npx stylus -I src/css/ < src/css/style-default.styl > static/css/style-default.css
$ npx stylus -I src/css/ < src/css/style-dark.styl > static/css/style-dark.css
$ npx stylus -I src/css/ < src/css/style-light.styl > static/css/style-light.css
# or
$ npx stylus -I src/css/ src/css/style-default.styl --out static/css/style-default.css
$ npx stylus -I src/css/ src/css/style-dark.styl --out static/css/style-dark.css
$ npx stylus -I src/css/ src/css/style-light.styl --out static/css/style-light.css
```
